# WTC Practice

A small browser tool for practicing Bach's Well-Tempered Clavier: load a prepared
prelude or fugue, isolate a voice, simplify the harmony, focus a passage, highlight
the fugue subject, and run a card-based warm-up. Scores are public domain.

**Live:** https://rebeccahwa.github.io/wtc-practice/

This repository holds the compiled static site.
